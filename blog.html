Hex VR Patch Notes | Minor Update

Version B0.6.488 - 6/23/2024

Communication & Clarity

Developer Notes:

Don't let the "Minor Update" tag in the title fool you, we've been hard at work improving the game in ways that matter a ton. This update includes bug fixes, quality of life improvements, and development tools that should make future development all that much smoother. -Blü

New Features:
Tutorial 1.0
The Main Menu now features a very simple text and image based tutorial explaining basic inputs and interactions
Score Reset Button
The Stadium Spawn Room Pedestals now feature a Score Reset button. This will set the room's score back to 0:0
Self Mute Toggle
The Personal Menu, now named the "Runic Slate", now features an audio setting for muting your own microphone
Mute Other Toggle
The Runic Slate's player list now has the ability to mute specific players
Mic Activity Indicator
When speaking, an icon will appear above your head to indicate that your microphone is active
HUD Settings
The Runic Slate's Graphics settings tab now features HUD customization options, allowing you to cycle through 3 different HUD options: Full, Minimal, and Off

Improvements:

Head Collider Improvements
Due to player feedback, the collider size of the player's head has been reduced by roughly 30%
Spectator Improvements
Spectators can no longer interact with the disk, and have had their visuals enhanced
Disk Handling Improvements
We've tweaked how the disk reacts to your movements based on player feedback. The wrist now has slightly less influence over the end throw direction, and throwing as a whole has been slowed down slightly. We'll be monitoring these changes closely
Disk Behavior Improvements
The disk should now retain less speed when bouncing, making it far more predictable. Speed retained reduced from 90% to 50%
Stadium Collision Improvements
Particularly troublesome areas of the Stadium have had their collision tweaked and smoothed out. Examples include the spawn tunnels and the side walls
Runic Slate Improvements
The Runic Slate is now partially visible to other players. This should help better communicate to others when you're occupied in a menu
Networking Improvements
Player position smoothing has been tweaked
Graphical Improvements
Multiple textures and materials have been improved, moving them closer to their final designs
Audio Improvements
Sound attenuation
Performance Improvements
Framerates across the board should see marginal improvements

Bug Fixes:

Remote Grab
The ability to extend your reach by dragging with 2 hands has been removed
Ghost Grab
The exploit allowing players to grab the air has been patched out
Runic Slate Summon Animation Fixed
The animation that plays when summoning the Runic Slate previously positioned the slate incorrectly if spawned while moving. This has been fixed
