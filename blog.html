Hex VR Patch Notes

Quality of Life | Major Update

Version B0.6.XXX - 5/XX/2024

Developer Notes:

Our first post-beta launch update! We've been hard at work over the past few weeks working on bug fixes, performance improvements, new quality of live (QOL) features, and disk handling improvements. We're excited for everybody to get their hands on these changes, and appreciate all of the feedback we've received thus far! A few of the new features and bug fixes that are in the works won't be featured in this update, as they need a little more time in the oven; rest assured we've received your feedback and are working on it! - Blü

New Features:
  • PCVR is Back!
    - Due to technical issues, the PCVR version of Hex VR was temporarily removed for download. This has been rectified, and is now available once again to download on SideQuest. The HMDs officially supported for PCVR remains restricted to Meta headsets.
  • Self Mute Toggle
    - This can be found on page 2 of the "Audio" settings
  • Mute Others Toggle
    - This can be found next to player names in the personal menu
  • Microphone Activity Indicator
  • Score Reset Button
    - The Stadium Pedestal now features a button that resets the lobby's score
  • Basic Inputs Tutorial
Improvements:
  • Disk Handling Improvements
    - Made several changes to throwing based on player feedback. Influence of the wrist has been reduced, and the influence of the arm has been reduced slightly. These changes will result in the disk feeling a little heavier, and a little more stable
  • Disk Behavior Improvements
    - Made several changes to disk behavior based on player feedback. The disk should preserve less velocity after bouncing
  • Performance Optimizations
    - Framerates across the board should see marginal improvements
  • Networking Improvements
    - Player positional smoothing has been tweaked. We'll continue to monitor these changes, as they still need additional work
  • Microphone Sensitivity Improvements
  • Player Display Name Filtering
    - Player Display Names now have profanity filtering. If you come across someone with an offensive Display Name, please report it to staff by DMing Hex-Modmail#9221 on Discord with details including the Display name of the offending Player, as well as the approximate date & time of the encounter
  • Spectator Mode Improvements
    - Spectators are now transparent, and cannot be grabbed by other players
  • Improved Player Collision
  • Improved Stadium Collision
  • UI Improvements
  • Graphical Improvements
  • Numerous Bug Fixes
Bug Fixes:
  • "Air Grabbing" has been removed
    - A bug allowed Players to grab the air as though they were grabbing a solid surface and gain an amount of speed greater than intended. This has been patched. As fun as it is, it's by far the most reported issue in game currently, and is widely considered greatly disruptive
  • "Remote Grabbing" has been removed
    - Previously, Players were able to pull themselves away from a surface while maintaining a grab over distances that were unintended. This has been patched.
  • Personal Disk no longer collides with the main Stadium Disk

Beta Release | Major Update

Version B0.5.372 - 3/29/2024

Developer Notes:

Hex VR is Officially in Open Beta! We are excited to introduce the first of many Beta updates to come for Hex VR. We are excited to see the community grow and evolve as we continue to develop the game. - Blü

New Features:
  • In Game Server Browser
  • Display Name System
  • Full Body Avatar IK
  • Team System
  • Regrabbing (Experimental*)
  • Pitch and Roll (Experimental*)
  • Spectator Mode (Experimental*)
  • App Lab

Experimental features are highly buggy and unfinished, but contribute to gameplay enough to justify releasing in an early state*

Improvements:
  • Menu Improvements
  • Improved Multiplayer Stability
  • Improved Movement
  • Improved Disk Handling
  • Improved Performance Optimizations
  • Improved Settings Menu
  • Improved Arena Geometry
  • Audio Improvements
  • Voice Chat Improvements
  • Graphical Improvements
  • Increased Total Player Cap
  • Numerous Bug Fixes
Bug Fixes:
  • Disk no longer "slingshots" when being pulled through geometry
  • Fixed CV1 throwing
  • Fixed Settings Menu "up" buttons not working
  • Eye animation will no longer stutter and skip
  • Player HUD no longer clips through geometry
