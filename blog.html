Hex VR Patch Notes

Quality of Life | Major Update

Version B0.6.XXX - 5/XX/2024

Developer Notes:

Our first post-beta launch update! We've been hard at work over the past few weeks working on bug fixes, performance improvements, new quality of live (QOL) features, and disk handling improvements. We're excited for everybody to get their hands on these changes, and appreciate all of the feedback we've received thus far! A few of the new features and bug fixes that are in the works won't be featured in this update, as they need a little more time in the oven; rest assured we've received your feedback and are working on it! - Bl√º

New Features:
PCVR is Back!
Due to technical issues, the PCVR version of Hex VR was temporarily removed for download. This has been rectified, and is now available once again to download on SideQuest. The HMDs officially supported for PCVR remains restricted to Meta headsets.
Self Mute Toggle
This can be found on page 2 of the "Audio" settings. Players can now mute themselves in-game, giving them more control over when they want to be heard.
Mute Others Toggle
This can be found next to player names in the personal menu's player list. Allows you to mute others for disruptive voice chat.
Microphone Activity Indicator
Turn Mode Settings
This can be found on page 1 of the "Input" settings. Despite there being an in game option for it already, changing the turn mode to "Snap" didn't do anything. This has been corrected.
HUD Settings
This can be found on page 1 of the "Graphics" settings. Different HUD modes are now available to allow the player to find their own balance between visual clutter and information.
Score Reset Button
The Stadium Pedestal now features a button that resets the lobby's score.
Basic Inputs Tutorial
The main menu now features a "Tutorial" button. When pressed, it displays a series of images that should hopefully help new players get familiar with the input scheme

Improvements:

Disk Handling Improvements
Made several changes to throwing based on player feedback. Influence of the wrist has been reduced, and the influence of the arm has been reduced slightly. These changes will result in the disk feeling a little heavier, and a little more stable.
Disk Behavior Improvements
Made several changes to disk behavior based on player feedback. The disk should preserve less velocity after bouncing.
Performance Optimizations
Framerates across the board should see marginal improvements.
Networking Improvements
Player positional smoothing has been tweaked. We'll continue to monitor these changes, as they still need additional work.
Microphone Sensitivity Improvements
The threshold for picking up Microphone activity has been raised slightly. This should reduce background noise pickup
Player Display Name Filtering
Player Display Names now have profanity filtering. If you come across someone with an offensive Display Name, please report it to staff by DMing Hex-Modmail#9221 on Discord with details including the Display name of the offending Player, as well as the approximate date & time of the encounter.
Spectator Mode Improvements
Spectators are now transparent, and cannot be grabbed by other players.
Improved Player Collision
Players should be slightly easier to grab, and the head collision volume used for headbutting has been reduced by 30%
Improved Stadium Collision
Multiple problematic areas of the Stadium have had their collision adjusted for gameplay purposes. We will continue to monitor these changes
UI Improvements
Some menus have been streamlined. In addition, the player HUD has received a minor visual overhaul
Graphical Improvements
Multiple assets have been improved including Stadium textures, Armor Textures, and Particle Effects
Lighting Improvements
Improved lighting fidelity while also reducing memory consumption
Audio Improvements
Sound attenuation in the Stadium has been tweaked. This will result in slightly less echoey audio

Bug Fixes:

"Air Grabbing" has been removed
A bug allowed Players to grab the air as though they were grabbing a solid surface and gain an amount of speed greater than intended. This has been patched. As fun as it is, it's by far the most reported issue in game currently, and is widely considered greatly disruptive.
"Remote Grabbing" has been removed
Previously, Players were able to pull themselves away from a surface while maintaining a grab over distances that were unintended. This has been patched.
Personal Disk no longer collides with the main Stadium Disk
Previously, the Player's Personal Disk was capable of colliding with the main Stadium disk during certain conditions. This has been patched; sorry ping-pong enjoyers
Fixed a bug that would cause the Player's Personal Menu to lag behind if summoned while moving
