Hex VR Patch Notes | Major Update

Version B0.7.XXX - 1/3/2025

A Very Merry Hex VR Update

Developer Notes:

Happy Holidays Hex Knights! This is Hex VR's first major update since the start of Beta and we're happy to share what we've been working on. Between new features, improvements to existing features, and some extra goodies, this Holiday update will be something to remember! - Blü

Seasonal Changes:
Winter's Here
The Main Menu and Stadium have been reflavored to incorporate a Winter/Holiday theme.
Seasonal Music
Enjoy the sounds of the season with the new Holiday Remix of the Main Menu music.
Festive Attire
The Hex Knights have been temporarily decked out in Festive hats.
Holiday themed changes will be removed sometime in late January!
New Features:
Gamemode 1.0 (Experimental)
This marks the introduction of our first iteration of internal game rules that we'll be sharing with you all. The room owner can start and end the match from the Stadium Pedestal in either spawn room at any time. Games are played first to 20 points.
Announcer 1.0 (Experimental)
We're happy to introduce the announcer of Hex VR's Stadium Gamemode, Freyliana (AKA, JeepGirlz)! At least that's what I'd like to say. Due to some timing constraints, Gamemode 1.0 will temporarily feature CadeXR's voice.
Stadium Reset
Room owners now have access to a new option in the Stadium Pedestal that allows them to reset the state of the arena, i.e. reset the disk position and team scores.

Improvements:

Personal Pedestal Rewrite
The Arcane Slate (Personal Menu) has been retooled into the Personal Pedestal. It's now accessible via the spawn rooms.
Personal Pedestal Improvements
All new layout and features included in the Personal Pedestal, including a small personal mirror.
Improved Stadium Audio Reverb Zones
Adjusted reverb zones for clearer audio.
Increased Voice Volume Multiplier
Even at maximum volume, player voices tended to be quite quiet. So we've increased the player voice attenuation range and increased the player voice chat volume multiplier from 1x to 1.5x.
Improved Visuals
Adjusted lighting and textures across multiple areas of the Stadium and Main Menu.
All New Toon Shader
Replaced the default Unity PBR Lit material with a custom built toon shader.
Background Systems
Created/Implemented several new systems that we'll be leveraging in new features coming in the near future including scene switching,

Bug Fixes:

Room Crash
Fixed an issue that caused rooms to crash and burn at 6 players. Will continue to monitor.
Disk's Meteor Shot "Fireball" Effect
Fixed an issue where a Disk's Meteor Shot "Fireball" particle effect wasn't being networked properly.
Disk Team Color
Fixed an issue where the Disk's Color, which indicates team ownership, wasn't being networked properly.
Reconnect Score Desync
Fixed an issue where leaving and rejoining a room would cause the room's score to become desynced.
Tutorial Button Colliders
Fixed Tutorial button colliders not being aligned to the button visuals properly.
Headless Reconnect
Fixed an issue where when reconnecting, other player's head positions weren't synced properly

Hex VR Patch Notes | Minor Update

Version B0.6.488 - 6/23/2024

Communication & Clarity

Developer Notes:

Don't let the "Minor Update" tag in the title fool you, we've been hard at work improving the game in ways that matter a ton. This update includes bug fixes, quality of life improvements, and development tools that should make future development all that much smoother. -Blü

New Features:
Tutorial 1.0
The Main Menu now features a very simple text and image based tutorial explaining basic inputs and interactions
Score Reset Button
The Stadium Spawn Room Pedestals now feature a Score Reset button. This will set the room's score back to 0:0
Self Mute Toggle
The Personal Menu, now named the "Runic Slate", now features an audio setting for muting your own microphone
Mute Other Toggle
The Runic Slate's player list now has the ability to mute specific players
Mic Activity Indicator
When speaking, an icon will appear above your head to indicate that your microphone is active
HUD Settings
The Runic Slate's Graphics settings tab now features HUD customization options, allowing you to cycle through 3 different HUD options: Full, Minimal, and Off

Improvements:

Head Collider Improvements
Due to player feedback, the collider size of the player's head has been reduced by roughly 30%
Spectator Improvements
Spectators can no longer interact with the disk, and have had their visuals enhanced
Disk Handling Improvements
We've tweaked how the disk reacts to your movements based on player feedback. The wrist now has slightly less influence over the end throw direction, and throwing as a whole has been slowed down slightly. We'll be monitoring these changes closely
Disk Behavior Improvements
The disk should now retain less speed when bouncing, making it far more predictable. Speed retained reduced from 90% to 50%
Stadium Collision Improvements
Particularly troublesome areas of the Stadium have had their collision tweaked and smoothed out. Examples include the spawn tunnels and the side walls
Runic Slate Improvements
The Runic Slate is now partially visible to other players. This should help better communicate to others when you're occupied in a menu
Networking Improvements
Player position smoothing has been tweaked
Graphical Improvements
Multiple textures and materials have been improved, moving them closer to their final designs
Audio Improvements
Sound attenuation
Performance Improvements
Framerates across the board should see marginal improvements

Bug Fixes:

Remote Grab
The ability to extend your reach by dragging with 2 hands has been removed
Ghost Grab
The exploit allowing players to grab the air has been patched out
Runic Slate Summon Animation Fixed
The animation that plays when summoning the Runic Slate previously positioned the slate incorrectly if spawned while moving. This has been fixed
